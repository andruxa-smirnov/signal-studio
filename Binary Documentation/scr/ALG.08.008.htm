Go To Option

The following operations can be performed:

  • To move the audio cursor to the first sample frame, use "Edit ->Go To -> First Frame" or "Ctrl+Home" (Keyboard Shortcut).
  • To move the audio cursor to the last sample frame, use "Edit -> Go To -> Last Frame" or "Ctrl+End" (Keyboard Shortcut).
  • To scroll as needed to make the audio cursor visible, use "Edit -> Go To -> Cursor" or "Ctrl+\" (Keyboard Shortcut). 
  • To scroll as needed to make the selection start visible, use "Edit -> Go To -> Selection Start" or "Ctrl+[ " (Keyboard Shortcut) .
  • To scroll as needed to make the selection end visible, use "Edit -> Go To -> Selection End" or "Ctrl+]" (Keyboard Shortcut).
